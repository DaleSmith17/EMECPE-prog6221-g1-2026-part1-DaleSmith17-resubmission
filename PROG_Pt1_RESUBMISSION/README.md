## Project Description
This project is a C# console-based cybersecurity awareness chatbot. It plays a voice greeting, displays an ASCII art title, asks for the user’s name, and answers basic cybersecurity questions.

## Features
- Voice greeting using WAV audio
- ASCII art display
- Personalised user greeting
- Cybersecurity responses
- Input validation
- Coloured console interface
- GitHub Actions CI

## Files Included
- Program.cs
- ChatbotApp.cs
- AudioPlayer.cs
- AsciiArtDisplay.cs
- UserInput.cs
- BotResponses.cs
- ConsoleDesign.cs
- greeting.wav

## How to Run
1. Open the project in Visual Studio.
2. Build the solution.
3. Make sure greeting.wav is in the project root.
4. Run the program.
